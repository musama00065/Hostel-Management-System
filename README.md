# 🏨 Hostel Management System

A **role-based Hostel Management System** built using **Core PHP and MySQL** to digitize and simplify hostel operations for administrators and students.

The system provides separate **Admin and Student dashboards** with a clean UI and structured workflow for managing hostel rooms, bookings, and student records efficiently.

---

## 🚀 Features

### 👨‍💼 Admin Panel
- Manage rooms and seat allocation
- Add / update / delete student records
- Monitor hostel bookings
- View system activity logs
- Dashboard with overview statistics

### 🎓 Student Panel
- Book hostel rooms
- View room details
- Manage personal profile
- View assigned room information

## ⚙️ Installation & Setup

1. Install **XAMPP**
2. Clone the repository:
   ```bash
   git clone https://github.com/musama00065/Hostel-Management-System.git
   ```
3. Move project to:
   ```
   htdocs/
   ```
4. Import database:
   - Open phpMyAdmin
   - Import `.sql` file from `/database`

5. Run project:
   ```
   http://localhost/Hostel-Management-System
   ```
